# Valentine Proposal

A fun Valentine's Day proposal app where the "No" button runs away from your cursor. Built with React, TypeScript, and Tailwind CSS.

**Live Demo**: https://codebywaqas.github.io/valentine-proposal/

## Features

- The "No" button teleports across the screen when you try to click it
- Screen shakes and guilt-trip messages appear
- The "Yes" button keeps growing bigger
- Confetti and celebration animation when you click "Yes"
- Floating hearts background and cursor trail effects

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
# Clone the repository
git clone https://github.com/CodeByWaqas/valentine-proposal.git

# Navigate to the project directory
cd valentine-proposal

# Install dependencies
bun install

# Start the development server
bun run dev
```

## Deployment

This project is automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
